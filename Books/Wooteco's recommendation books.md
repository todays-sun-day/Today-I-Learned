# 📗 안드로이드 학습에 도움되는 책 정리
우테코 프리코스를 끝내며, 학습 로드맵을 다시 보게 되었다.  
앞으로 안드로이드를 학습하기 전에 도움이 되는 책을 학습하면 좋겠다는 마음에, 우테코 디스코드에 올라온 글을 다시 한 번 이곳에 작성하려고 한다.

### 소프트웨어 장인
개발자로서 어떤 마음가짐으로 살아갈 것인지에 대한 가이드를 제시하는 책.
### 1만 시간의 재발견
의식적인 연습이 무엇이며, 의식적인 연습이 가능하도록 설계하는 방법에 대해 다룬 책.
### 함께 자라기
내가 성장하는 방법, 우리가 함께 성장하는 방법, 매일매일 성장하는 방법에 대해 다룬 책.
### 좋은 코드, 나쁜 코드
나쁜 코드에 대해 이유를 설명하고, 좋은 코드로 바꿔가는 과정을 볼 수 있는 책.
### Clean Code (클린 코드)
클린 코드를 구현하기 위한 다양한 규칙을 설명한 책.
### 테스트 주도 개발 시작하기
테스트 주도 개발을 하는 법을 알려주는 책.
### 테스트 주도 개발
TDD의 구체적 사례와 패턴을 제시하는 책.
### 리팩터링 2판
리팩터링 개념, 리팩터링이 필요한 이유와 다양한 리팩터링 카탈로그를 제공하는 책.
### 객체지향의 사실과 오해
객체지향이란 무엇인가라는 원론적 면에서도 다소 위험한 질문에 답하기 위해 쓰인 책.
### 코틀린 인 액션 2판
코틀린 언어를 개발한 젯브레인의 코틀린 컴파일러 개발자들이 직접 쓴 일종의 공식 책.
### 코틀린 쿡북
개발 실무 중에 마주치는 여러 상황에서, 간결하고 안전하며 좀 더 코틀린스럽게 코드를 짜는 방법을 제시해준 책.
